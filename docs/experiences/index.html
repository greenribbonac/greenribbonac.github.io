臓器移植はまだ身近に感じることができるほど事例は多くなく、自分事のように考えるのは難しいかもしれません。
臓器移植を行わなければ遠からぬ死を迎えていたかもしれない人が、誰かからいのちをもらい“今”を生きている。
私たちはその事実を、臓器移植という経験を、活動を通じて一人でも多くの人に伝えていきたいと考えています。
