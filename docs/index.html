「移植医療」で助かるいのちを一つでも多く救いたい。
私たちはそんな想いから、移植医療の普及啓発のため活動している市民ボランティア団体です。
あなたの意思が誰かの命を、未来を救うかもしれません。
まずは私たちと一緒に「移植医療」について考えてみませんか？
