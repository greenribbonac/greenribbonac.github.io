『市民ボランティアによる移植医療の普及啓発活動の輪を日本全国に広めたい！』
そんな熱い想いを持ちながら、現在、主に東京、大阪、広島で市民ボランティアによる地域に根ざしたイベント運営を行っています。今後、このような活動を全国に広めていき、一人でも多くの方に移植医療に知ってもらうため、どうか活動のご支援をお願いいたします。ご支援いただいた団体様、個人様は当協会HPにてご紹介いたします（匿名も可）。
